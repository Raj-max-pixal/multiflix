<p>
<strong>Multiflix</strong> is a modern, Netflix-inspired movie discovery web application built to demonstrate
real-world frontend engineering, clean architecture, and premium UI thinking.
The platform allows users to explore movies across multiple categories using YouTube trailers
as the content source.
</p>

<p>

This project is designed as a portfolio-grade application suitable for internships, placements,
